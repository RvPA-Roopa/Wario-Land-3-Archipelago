# Wario Land 3 — Transformation Logic Reference

> Rules for **Transformation Shuffle = ON**. Form items serve as OR-alternatives to vanilla abilities. If shuffle is off, Form checks are ignored and logic falls back to the vanilla ability requirement.
>
> "Vampire" = Progressive Vampire (1st copy). "Bat" = Progressive Vampire (2nd copy).

## Summary: Where Each Form Helps

- **Puffy** — reach high places across most levels (alternative to High Jump Boots / Bat)
- **Bouncy** — same role as Puffy in most cases
- **Vampire / Bat** — Bat covers high-jump/float logic throughout; Vampire is required in E3 Red Key and in fire-torch chains (W6/E6)
- **Fat** — weight / ground-pound alternative (equals Overalls T1 in most rules; a few rules require specifically Fat or Overalls T2)
- **Snowman** — pairs with Fat as an Overalls T1 / ground-pound alternative
- **Zombie** — drop-through-floor alternative (N5 Red Key, N6/S4 floor-break chains, E6/E7 niches, torch-lighting)
- **Fire** — torch-lighting (W6, E6) + fire-drench chain in W4/E6
- **Yarn** — Grab alternative in W5 Beneath the Waves, Overalls T1 alternative in W6 West Crater (grey & red key)
- **Ice Skatin'** — W4 Green Chest only (one of several Grab×2 alternatives)
- **Flat** — W2 Red Key only (alternative to Truck Wheel)
- **Invisible** — no logic role; pure flavor

---

## Note on Progressive Vampire

Logic uses `state.count("Progressive Vampire") >= 2` to gate Bat-required locations. The 2nd copy is always placed after the 1st is reachable. In receipt, the client always applies Vampire (bit 1) before Bat (bit 6) regardless of collection order.
